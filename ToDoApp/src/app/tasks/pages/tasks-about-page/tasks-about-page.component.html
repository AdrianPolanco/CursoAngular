Este proyecto trata de un gestor de tareas simple, desarrollado con Angular. En la aplicación se pueden realizar las siguientes acciones:
  • Crear tareas
  • Ver tareas
  • Ver descripción detallada de una tarea
  • Editar tareas
  • Eliminar tareas
  • Marcar tareas como completadas
  • Marcar tareas como pendientes
  • Marcar tareas como en progreso
  • Marcar tareas como caducadas
